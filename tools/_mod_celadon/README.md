# Инструменты для модификации Celadon

Тут будет информация о всех инструментах созданных для облегчения процесса кодинга, спрайтинга и мапинга. Это каталог. Более подробно о инстрмуентах можно почитать в ридми к каждому из них.

- `extract_dmi`		- инструмент позволяет извлекать из dmi все спрайты в нужный формат картинки
- `parallax_tester`	- инструмент позволяет тестировать и эмуллировать паралакс в игре
- `Profiler_Parser`	- инструмент созданный для распарсирования (расшифровки) логов профайлера
