### `parallax_tester/` - Тестер паралакса
**Что делает:** Позволяет тестировать комбинации слоев паралакса в браузере

**Как использовать:**
1. Помести папку с извлеченными спрайтами в `parallax_tester/`, ну то есть сюда рядом, для этого потребуестя запсутить скрипт по извлечению спрайтов паралакса. А также добыть сам dmi файл паралакса. Ты справишься
2. Открой `parallax_viewer.html` в браузере
3. Нажми "Setup Layers" для настройки
4. Выбирай спрайты для каждого слоя, разберешься там немного кнопок
5. Настраивай альфу, режимы наложения, скорость, можешь рандомить результат
6. Смотри результат в статичном и анимированном предпросмотре. Только не залипни, на радуги откисни

**Файлы:**
- `parallax_viewer.html` - основной интерфейс

## Список файлов что нужно собрать
```
layer1/ (звезды):
Layer 1: layer1_0, layer1_1, ... layer1_20 (21 элемент)

layer2/ (туманности):
Layer 2: layer2_0, layer2_1, ... layer2_20 (21 элемент)

layer3/ (передний план):
Layer 3: layer3_0, layer3_1, ... layer3_20 (21 элемент)

layer4/ (объекты):
Layer 4: layer4_0, layer4_1, ... layer4_20 (21 элемент)
```

Это жестко заданный список. Любой из файлов ты можешь заменить на свою картинку, главное чтобы имя было такое же!

---

## Быстрый старт

```
parallax_tester/ → помести папку extracted_parallax → открой parallax_viewer.html
```

## Требования

- **Python 3.x** с библиотекой Pillow
- **Современный браузер** (Chrome, Firefox, Edge)
- Для работы в браузере может потребоваться локальный сервер: `python -m http.server` (но обычно нет)
- Паралакс тестер работает с 4 слоями: Stars, Nebulae, Foreground, Objects. Все статично и жестко задано
- Но если надо, свои новые комбинации попробовать захочешь, то замени то что есть в папках на свои картинки. О да я вижу, как ты начинаешь смотреть на вайфу свою в движении
- Ещё один момент, картинки тут 480х480 размером. Мб появится более лучший потом формат
- Папка не содержит всех паралаксов, потому что все слои будут очень тяжелыми для клиента
