### `extract_dmi/` - Извлечение спрайтов из DMI
**Что делает:** Конвертирует DMI файлы в отдельные изображения, вытаскивает спрайты коротко если

**Как использовать:**
1. Поместить DMI файл в папку `extract_dmi/` то есть сюда рядом со скриптом!
2. Запустить `extract_parallax.bat`
3. Выбери режим именования (0-2) какой те надо. Не спрашивай почему отчет идет с 0
4. Выбери формат файлов (PNG/JPG/GIF/BMP), помни гиф файл формат он тупо раскладывает на составляющие на кадры анимацию. Ну в принципе удобно
5. Получи спрайты в папке `extracted_[имя_файла]/`, имя выбирается по имени атласа что ты грузить будешь в скрипт.

**Файлы:**
- `extract_parallax.bat` - запуск программы
- `extract_dmi_with_names.py` - основной скрипт

## Быстрый старт

```
extract_dmi/ → помести parallax.dmi → запусти extract_parallax.bat
```

## Требования

- **DMI файлы** из игры для извлечения
- DMI файлы должны быть в формате PNG (стандарт BYOND)
