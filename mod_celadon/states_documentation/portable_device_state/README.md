# portable_device_state

Универсальное состояние UI для портативных устройств, позволяющее использовать интерфейс в лежачем положении.

## Использование

```dm
/obj/item/your_device/ui_state(mob/user)
	return GLOB.portable_device_state
```

## Логика

- Проверяет наличие предмета в инвентаре пользователя
- Проверяет сознание пользователя (`user.stat == CONSCIOUS`)
- Разрешает использование UI в лежачем положении

## Подходит для

- Рации, PDA, планшеты
- Портативные сканеры и устройства
- Любые портативные устройства с UI

## НЕ подходит для

- Стационарные устройства (компьютеры, консоли)
- Устройства, требующие активного взаимодействия

## Миграция

```dm
// Было:
/obj/item/device/ui_state(mob/user)
	return GLOB.inventory_state

// Стало:
/obj/item/device/ui_state(mob/user)
	return GLOB.portable_device_state
```

## Отладка

- **UI неактивен для лежачих**: Убедитесь, что используете `GLOB.portable_device_state`
- **UI активен без предмета**: Проверьте, что предмет в инвентаре пользователя
- **UI активен для без сознания**: Проверьте `user.stat == CONSCIOUS`
