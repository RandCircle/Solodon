### Описание мода

Этот модуль добавляет возможность переключать отображение обуви между обычным слоем (поверх одежды) и слоем под одеждой.

- **Ctrl+ЛКМ** по обуви переключает режим отображения
- В режиме "под одеждой" обувь отображается на слое чуть ниже одежды (UNIFORM_LAYER - 0.1)
- В обычном режиме обувь отображается на стандартном слое (SHOES_LAYER)
- Состояние сохраняется для каждого предмета обуви индивидуально
- Информация о текущем режиме отображается при осмотре обуви
